# 📌 Overview

This project was developed as part of a Machine Learning course. The main objective is to study and implement learning algorithms, with a focus on tabular methods and experimental analysis.

The project combines:

- Theoretical explanations  
- Environment simulation  
- Model implementation  
- Experimental evaluation  

---

# 🎯 Objectives

- Understand the fundamentals of the problem formulation  
- Implement and analyze learning algorithms (e.g., Q-Learning, SARSA)  
- Evaluate model performance under different conditions  
- Explore the relationship between theory and practical results  

---

# 🧠 Project Structure

```bash
ml-project/
│
├── notebooks/        # Jupyter notebooks (analysis and experiments)
├── src/              # Core implementation (environment, models, training)
├── data/             # Datasets (if applicable)
├── results/          # Outputs, metrics, and saved results
│
├── README.md
├── requirements.txt
└── .gitignore
````

---

# 📓 Notebooks

* `01_eda.ipynb` → Data exploration and initial analysis
* `02_environment.ipynb` → Problem definition and environment description
* `03_models.ipynb` → Theoretical explanation and model implementation
* `04_experiments.ipynb` → Training and experimentation
* `05_results.ipynb` → Final results and conclusions

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/victorbahh/Machine-Learn-Project.git
cd ml-project
```

Install dependencies:

```bash
pip install -r requirements.txt
```
